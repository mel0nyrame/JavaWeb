# Ajax

并非新的技术而是一种js的技术规范

发送请求的方式

- 在浏览器中输入地址
- 自动的请求:img/bmp等资源的获取
- 超链接
- 运行js代码产生请求,通过事件触发js代码,再动态判断

Ajax的最大优点就是能不加载整个页面的情况下与服务器交换数据并更新页面

同步交互和异步交互

- 同步交互
  - 服务端等待响应,客户端发送请求,服务端接收并处理请求,客户端等待响应,客户端需要等待服务端的响应才能做出相对于的行为
- 异步交互(使用XmlHttpRequest)
  - 客户端可以不必等待服务端响应继续操作

Ajax实现方式:

- 原生js(繁琐)
- jquery(已经封装好的)
- Vue中的axios(框架)
