# 前端工程化

将前端相关的代码剥离出来,形成一个独立的工程,并且使用软件工程的方式来解决开发过程中的问题

技术栈

- ES6:js的新语法
- Nodejs:前端运行环境
- npm:依赖下载工具
- vite:前端构建工具
- router:通过路由实现页面切换
- pinia:通过状态管理实现组件数据传递
- axios:ajax异步技术封装

## npm相关命令

- npm init:初始化项目
- npm install 包名@版本号(npmjs.com可以查找版本号)
  - npm install -g 包名:安装全局依赖
  - npm install 安装package.json中记录的所有依赖
- npm update 包名:升级依赖
- npm uninstall 包名:卸载依赖
- npm ls:查看项目依赖
  - npm ls -g:查看全局依赖
- npm run:运行项目脚本(看package.js中的scripts写的什么)
